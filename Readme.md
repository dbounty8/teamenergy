# Energy Application

Energy app using Node.js, Express and MongoDB and React.

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Server

```bash
npm start
```

You can browse the apis at <http://localhost:3000>

## Irradiation Data info
